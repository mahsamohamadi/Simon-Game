# Simon Game

A simple and fun browser-based Simon Game built with HTML, CSS, and JavaScript. Follow the sequence of flashing colors and sounds, then repeat the pattern correctly to advance to the next level. With each round, the sequence grows longer and the challenge increases.

## Features

* Smooth color and sound animations
* Increasing difficulty
* Responsive design
* Simple and clean UI

## How to Play

Press any key to start, watch the pattern carefully, then click the buttons in the same order. If you make a mistake, the game resets and you can try again.

## Technologies Used

* HTML
* CSS
* JavaScript

Enjoy testing your memory and improving your skills!
